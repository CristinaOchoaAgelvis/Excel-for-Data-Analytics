## 📄 Descripción del Proyecto

Este proyecto forma parte de un análisis de Retail Business Intelligence, enfocado en la integración y preparación de datos comerciales de una red de tiendas minoristas. El objetivo es organizar, limpiar y estructurar información proveniente de clientes y ventas para facilitar su posterior análisis y generación de insights de negocio.

Durante el proceso se trabajó con los archivos Clientes.csv y Vendas.csv, realizando tareas de importación, estandarización de formatos, limpieza de datos y generación de cálculos iniciales. Estas acciones permiten asegurar la calidad de los datos y preparar la información para futuros análisis exploratorios y evaluaciones del rendimiento comercial.

El proyecto busca sentar las bases para analizar patrones de consumo, comportamiento de clientes y desempeño de las ventas dentro de la operación minorista.

## 🔑 Puntos Clave del Proyecto

Importación y organización de los datasets Clientes y Ventas.

Estandarización de formatos de datos (texto, número, moneda y fechas).

Limpieza de registros incompletos en el dataset de ventas.

Identificación de clientes únicos a partir de los registros de ventas.

Generación de columnas adicionales como bono de fidelización y ID de venta encriptado.

Exportación de los datos estructurados en formatos alternativos como PDF.
